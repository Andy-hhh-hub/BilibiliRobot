# B站直播机器人(BilibiliRobot)

我之前用树莓派搭建了一个监控我家狗的B站直播，https://live.bilibili.com/21172572　

我几年前做过一个微信机器人，接入图灵机器人的API，现在将它接入到Ｂ站直播弹幕中，用来消遣消遣，纯粹玩玩。

### 依赖

* Python 3.5+
* pip3 install aiohttp

### 快速开始

在 var_set.py 中配置参数．

在命令行中，

```python
python main.py
```
